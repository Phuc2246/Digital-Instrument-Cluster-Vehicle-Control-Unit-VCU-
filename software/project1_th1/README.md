# Digital Instrument Cluster & VCU (Qt/QML)

EV dashboard đa màn hình viết bằng Qt 5.15.2 (Qt Quick Controls 2).

## Build (qmake)
qmake project.pro
make -j4

## Cấu trúc (gợi ý)
src/         # C++
qml/         # QML (Dashboard1/2/3)
assets/      # images, icons, fonts
docs/        # screenshots, diagrams
